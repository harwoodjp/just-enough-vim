# Setup
  * Config file is `~/.vimrc`
  * Turn on syntax highlighting
    * `syntax on`
  * Set indent to 2 spaces
    * ```
      set expandtab
      set shiftwidth=2
      set softtabstop=2
      ```
# Navigation
  * Move to beginning of document
    * `gg`
  * Move to end of document
    * `G`
  * Move to beginning of word
    * Forward
      * `w`
    * Backward
      * `b`
  * Move to beginning of line
    * `0`
  * Move to end of line
    * `$`
  * Down a line
    * `j`
  * Up a line
    * `k`
  * Right a column
    * `l`
  * Left a column
    * `h`
  * Move to top of screen
    * `H`
  * Move to middle of screen
    * `M`
  * Move to bottom of screen
    * `L`
  * Search
    * `/[search-term]`

# Writing
  * Insert mode
    * `a`
  * Undo
    * `u`
  * Redo
    * `ctrl` + `r`
  * Select
    * `V` then move cursor
  * Cut
    * `d`
  * Yank/copy
    * `y`
  * Paste
    * `p`
  * Delete line
    * `dd`
  * Write/save
    * `w`
  * Quit
    * `q`
